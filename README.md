# Calculadora de Porcentajes Pro

Calculadora de Porcentajes Pro es una extensión gratuita para Firefox que permite realizar cálculos de porcentajes de forma rápida y sencilla.

## Funciones

- Calcular porcentajes
- Calcular descuentos
- Calcular IVA
- Calcular aumentos porcentuales
- Calcular disminuciones porcentuales
- Calcular cambios porcentuales

## Instalación

Disponible para Firefox.
